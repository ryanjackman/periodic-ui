## Periodic UI

This is the webview for a project I will be starting soon.

Each element of the periodic table is displayed on its own page will all details listed.

---

To run this project:

1. Clone this project
1. `npm install`
1. `npm run build`
1. `npm run start`
1. Navigate to localhost:3000
